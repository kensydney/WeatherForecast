# WeatherForecast

# Backend
  Open Visual studio 2015 
  
  Load weather.sln
  
  Run the app 
  
  Web api listening on 52166 port
  
# Backend testing
  Open Visual studio 2015 test exploer
  
  Run all the tests by click run all.
  
# Front-end 
  Node version 8.9.0
  
  NPM version 5.5.1
  
  Run npm install for all dependencies
  
  Run ng serve 
  
  Open browser on port 4200.
  
# Front-end testing
  Run ng test
